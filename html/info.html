Start Using Cloaq

1. Pin the extension by opening your browser's extensions menu (found by clicking the puzzle piece icon next to the address bar) and selecting the thumbtack icon.

2. Click the extension icon on your toolbar (top right of the browser).

3. Adjust the options to your desired values.

How Cloaq Works

Cloaq uses the chrome.debugger API to change data directly at the browser level, making it effective across all frames and web workers, unlike other extensions that rely on less reliable script injections. This approach ensures data modifications are undetectable and work consistently, even during the initial page load.

IP Address

Cloaq does not change your IP address. To change your IP address you will need a VPN or proxy.

Hide Debugging Notification Bar

While the extension is on, a notification bar becomes visible. Hiding the bar can be done by using the --silent-debugger-extension-api flag. Instructions on how to run chrome with flags.

Limitations

Credential Pages: Cloaq cannot spoof location or timezone on pages containing login forms, password fields, or other credential entry forms (such as Google sign-in, Facebook login, bank sites). Chrome security prevents any extension from debugging these pages to protect your sensitive data.

This is part of Chrome's security features designed to prevent cookie theft and credential stealing. If you see the debugger banner briefly appear and then disappear on such pages, this is expected behavior and not a bug.
